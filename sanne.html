Terug

Sanne 't Hooft

Sanne is de welbekende docent van het vak CSS bij de minor WDD. Een professional op het gebied van css die alles met wat lijntjes code kan illustreren.

Vanwege het missen van een Weekly Nerd heb ik deze ingehaald bij de zoom meeting van CSS cafe met als gastspreker Sanne. De presentatie ging over anchor positioning. Dit is een nieuw element waarbij we al geintroduceerd werden met de Hackathon. Hij begon de presentatie met: The best way to learn css is “doing stupid experiments”. Zo is hij tijdens de presentatie aan het live coderen om dus te spelen met wat er allemaal mogelijk is. Het interressante aan het coderen van Sanne is dat hij heel detailgericht werkt. Hij was bijvoorbeeld bezig met een 3 lampen die als radio buttons werken. daarbij liet hij een motje vliegen van de ene button naar de andere. Uiteraard was het overspringen met behulp van anchor positioning al voldoende maar toch besloot hij om de beweging hiervan soepel te laten verlopen om het plaatje compleet te maken.

Wat neem ik mee

Van Sanne heb ik al heel veel mogen leren tijdens de lessen maar ook met deze presentatie krijg je te zien hoe fijn het is om detail gericht te werken. De kleine bewegingen gepaard met mooie strakke styling geeft een perfecte uitstraling van een website.
